# Flex Sensor Research

There are a couple options for creating the 'flex sensor'. One is to buy a pre-made component that varies resistance based on the curve of the part. The other is to create our own DIY solution, which would involve using a piece of conductor taped onto the back of the finger. Similar hand gesture based projects online suggest that creating a DIY part is considerably more prone to error, inconsistencies, and misreading. Hence, for this project, we will opt for a pre-made flex sensor component. 

The most wide-spread and obvious intial choice is SpectraSymbol's 'Flex Sensor FS'. However, some further research suggests that there is a new and improved edition: SpectraSymbol's 'Flex Sensor FLX'. The FLX version has more extensive testing documentation. Compared to the older FS verison, it has improved repeatability, less signal drift, better flexibility and a lower profile. 

The most important specs of the SpectraSymbol Flex Sensor FLX are as below:
<ul>
  <li>55mm length and 0.13mm total thickness.</li>
  <li>12k ohm FLAT resistance.</li>
  <li>4x resistance change multiplier (i.e., resistance increases by up to 4x as bend increases).</li>
  <li>5VDC Input, 0.5W continuous and 1W peak power rating.</li>
  <li>+- 2% repeatability.</li>
</ul>

Note that we will specifically be using the FLX-L-055-123-MP. 055 simply represents the part length, and MP the 'male pin' edition.
